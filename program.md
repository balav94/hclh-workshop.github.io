# Program / Schedule

The workshop will feature invited talks, peer-reviewed paper presentations, and a panel discussion.

> This is a **tentative half-day schedule** (subject to change):

| Time | Session |
|---:|---|
| 09:00 – 09:10 | Welcome and Introductory Remarks (Organizing Committee) |
| 09:10 – 09:45 | Invited Talk I |
| 09:45 – 10:15 | Accepted Paper Session I (3 papers; 7 min + 3 min Q/A each) |
| 10:15 – 10:30 | Break |
| 10:30 – 11:05 | Invited Talk II |
| 11:05 – 11:45 | Accepted Paper Session II (4 papers; 7 min + 3 min Q/A each) |
| 11:45 – 12:15 | Panel Discussion |
| 12:15 – 12:30 | Concluding Remarks and Open Discussion |
